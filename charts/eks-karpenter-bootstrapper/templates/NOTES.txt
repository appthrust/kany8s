The bootstrapper will:
  - provision the EKS cluster's OIDC provider (iam.services.k8s.aws/OpenIDConnectProvider)
  - create the Karpenter controller IAM Role/Policy (iam.services.k8s.aws/Role, Policy)
  - create the node instance profile
  - create the cluster security group for Karpenter-managed nodes
  - install Karpenter via Flux HelmRelease{{- if .Values.args.karpenterChartVersion }} at version {{ .Values.args.karpenterChartVersion }}{{- end }}
